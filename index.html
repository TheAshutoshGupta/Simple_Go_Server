Lightweight

The server is tiny, with minimal dependencies and instant startup times for fast development feedback.

Secure Defaults

Strong headers, content-type detection, and simple TLS options make production safer with minimal config.

Live Reload

Auto-reload pages on file changes to accelerate the dev loop — no extra tooling required.

Custom Routing

Serve SPA fallbacks, custom rewrites, and static assets with a tiny routing table.

Metrics

Optional lightweight metrics endpoint for basic usage monitoring and health checks.

Cross-platform

Built with Go — works the same on Windows, macOS and Linux. Single binary deployment.

Quick examples

Simple serve

Serve a directory on port 8080.

go run main.go --dir=./dist --port=8080

SPA fallback

Route 404s to index.html for client-side routing.

--spa=true

Enable TLS

Quickly enable TLS with certificate and key files.

--tls-cert=cert.pem --tls-key=key.pem
